PROJECT DESCRIPTION

Project Aardvark was an idea created by our group with the goal to be able to decode audio files and manipulate them. Some aspects of manipulation we intended were to display the audio data visually and normalize the audio files. This type of manipulation is refered to as digital signal processing or DSP. DSP is characterized by the representation of discrete time, discrete frequency, or other discrete domain signals by a sequence of numbers of symbols and the processing of these signals.*

Project Aardvark will attempt to use the Java coding langauge as well as Linux Bash script to perform various manipulations of selected audio files.
